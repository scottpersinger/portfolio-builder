# Portfolio Builder

This repo contains some AI prompts/skills for generating a "personal portfolio" website.

A portfolio site is a great way to showcase your work, experience, and interests to potential employers.

## Claude code skill

The easiest way to create your portfolio is to use **Claude code**. 

Download the Claude code [skill](./create_portfolio) and use it to create your portfolio.

**Important** You should install and setup the [Claude-in-Chrome](https://chromewebstore.google.com/publisher/anthropic/u308d63ea0533efcf7ba778ad42da7390?pli=1) extension. This allows Claude to read web pages which
is critical for gather your portfolio information. The Chrome-dev-tools extension can work also.
Claude will  also use the extension to visually test your portfolio as it creates it.


## Hosting your portfolio

The portfolio can be hosted anywhere that hosts static HTML assets. One easy place is Github
itself, using Github pages.
