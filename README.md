# Portfolio Builder

This repo contains some AI prompts/skills for generating a "personal portfolio" website.

A portfolio site is a great way to showcase your work, experience, and interests to potential employers.

## Claude code skill

Download the Claude code [skill](./create_portfolio) and use it to create your portfolio.

The skill relies on having a browser automation MCP available (like Claude-in-chrome) where you have
already logged into Linkedin.
